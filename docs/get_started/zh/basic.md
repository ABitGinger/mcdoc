## 为什么不推荐中国版我的世界
中国版游戏本身免费，但是游戏更新相对国际版通常会落后*两三年*，并且限制较多，几乎不能使用外部的资源，只能通过网易自带的商店购买；不能加入外部服务器，只能在网易租赁服务器或加入网易合作服务器。
