    | 传送临时据点                | essentials.warp            | /warp 玩家名                            |
    | 传送到下线前的位置            | essentials.tpoffline       | /tpoffline 玩家名                        |

    |                          | essentials.tpahere        | /tpahere 玩家名（申请把玩家tp到自己的位置） |

    |                          | essentials.tpdeny         | /tpdeny 拒绝                            |
